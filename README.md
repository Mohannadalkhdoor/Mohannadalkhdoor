# مهند محمد الخضور | Mohannad Alkhdoor

## نبذة شخصية
- مشرف رقابة داخلية في سلطة المياه – عمّان، الأردن  
- مدير تأمينات سابق في عدة مستشفيات خاصة  
- خبرة في التأمينات الطبية والتدقيق الداخلي  
- مهتم بالصحة والتغذية والكتابة التحليلية عن القيادة والإدارة  
- تاريخ الميلاد: 8 أيلول 1987
